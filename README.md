# django-blog
> Blog Application using Python + Django

## Installation

1. Clone repo
1. `$ cd django-blog`
1. Setup virtualenv (Optional)
	1. `$ vitualenv venv`
	1. `$ source venv/bin/activate`
1. `$ pip install -r requirements.txt`
1. `$ python manage.py migrate`
1. `$ python manage.py runserver`
